<h1>mymenu.js</h1>
This file is to create expentable or collapsable menu using custom tag <my-menu>and<my-menu-header>

<h2>Html</h2>
```
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Title</title>
</head>
<body>
<my-menu expandState="expend">
    <my-menu-header>
        <div>Title</div>
    </my-menu-header>
    <p>It is a paragraph</p>
    <p>It is also a paragraph</p>
</my-menu>
<script src="../JS/jquery.js"></script>
<script src="../JS/MyJS/mymenu.js"></script>
</body>
</html>
```
